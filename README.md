# minfactory

minFactory

## Installation

```bash
cd client/ && npm install
cd server/ && npm install
```

## Start

```bash
cd client/ && npm start
cd server/ && npm start
```

## Test

```bash
cd client/ && npm test
cd server/ && npm test
```

## Lint
```bash
cd client/ && npm run lint
cd server/ && npm run lint
```

## Docs

```bash
cd client/ && npm run gen:docs
cd server/ && npm run gen:docs
```
